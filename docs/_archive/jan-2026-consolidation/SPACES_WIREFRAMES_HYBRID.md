### TerritoryQuadrant Component

```tsx
function TerritoryQuadrant({
  icon,
  name,
  spaces,
  totalCount,
  variant
}: TerritoryQuadrantProps) {
  const ref = useRef(null);
  const isInView = useInView(ref, { once: true, margin: '-100px' });

  return (
    <motion.div
      ref={ref}
      className="territory-quadrant"
      variants={variant}
      initial="initial"
      animate={isInView ? "animate" : "initial"}
      whileHover={{ scale: 1.02 }}
    >
      {/* Territory header */}
      <div className="flex items-center gap-2 mb-4">
        <span className="text-2xl">{icon}</span>
        <h3 className="text-sm font-medium text-white/60">{name}</h3>
      </div>

      {/* Space cards in mini-grid */}
      <div className="grid grid-cols-2 gap-3">
        {spaces.map((space, i) => (
          <motion.div
            key={space.id}
            custom={i}
            variants={spaceCardsInTerritory}
            initial="initial"
            animate={isInView ? "animate" : "initial"}
          >
            <SpaceCard space={space} variant="compact" />
          </motion.div>
        ))}
      </div>

      {/* Explore more link */}
      <Link
        href={`/spaces/browse/${name.toLowerCase()}`}
        className="mt-4 text-sm text-white/40 hover:text-[#FFD700]"
      >
        Explore {totalCount} more â†’
      </Link>
    </motion.div>
  );
}
```

## Variations to Consider

### Variation 1: Territory Map Hero

**Change:** Move territory map to above fold (Zone 1), your spaces below

```
Above fold: Territory map (all 4 quadrants)
Scroll 1: Your Spaces (detailed cards)
Scroll 2: Start Something
```

**Tradeoff:**
- âœ“ Spatial metaphor becomes the identity
- âœ— "Your spaces" gets deprioritized
- âœ— May be disorienting for new users

### Variation 2: Territory Deep Zoom

**Change:** Clicking quadrant zooms into it (animated), shows full space list

**Effect:**
- Click "UNIVERSITY" quadrant
- Other 3 quadrants fade out
- University quadrant expands to full width
- Shows all 47 spaces with filters
- Back button shrinks back to 2x2 grid

**Tradeoff:**
- âœ“ No route change needed
- âœ“ Maintains spatial continuity
- âœ— More complex state management
- âœ— Browser back button doesn't work

### Variation 3: Animated Territory Dividers

**Change:** Draw vertical/horizontal gold lines between quadrants after border reveal

```tsx
// After outer border completes
const dividers = {
  vertical: {
    initial: { scaleY: 0 },
    animate: {
      scaleY: 1,
      transition: { delay: 0.8, duration: 0.6, ease: [0.22, 1, 0.36, 1] }
    }
  },
  horizontal: {
    initial: { scaleX: 0 },
    animate: {
      scaleX: 1,
      transition: { delay: 0.8, duration: 0.6, ease: [0.22, 1, 0.36, 1] }
    }
  }
};
```

**Effect:** Creates window-pane effect, reinforces quadrant structure
